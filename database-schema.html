Core Infrastructure (public schema)

users

Registered users. Authentication + role separation.
PKUQ
id BIGSERIAL · username · email · password · role

data_sources

Every data source the system ingests from. Central reference table.
PKFK→users
id UUID · source_type · trust_score · connection_config JSONB · active

source_schemas

Versioned schema snapshots. New version on every schema drift.
PKFK→data_sources
id UUID · version · status (ACTIVE/SUPERSEDED)

source_schema_fields

Column metadata within a schema — type, format, sample values.
PKFK→source_schemas
id UUID · field_name · inferred_type · detected_format · sample_values JSONB
Ingestion Pipeline (public schema)

ingestion_jobs

One row per ingestion run. Status + per-stage record counters.
PKFK→data_sources
id UUID · status · total_read · total_transformed · total_loaded · error_message

staged_records

Raw source data landing zone. Everything passes through here first.
PKFK→ingestion_jobsFK→data_sources
id UUID · raw_payload JSONB · record_hash · row_number

field_mappings

Source field → canonical field with confidence. PENDING blocks data flow.
PKFK→data_sources
id UUID · canonical_entity · canonical_field · confidence · status · transform_rule
Profiling & Quality (public schema)

data_profiles

Statistical snapshot of source data — baseline for drift detection.
PKFK→data_sourcesFK→source_schemas
id UUID · total_records

field_profiles

Per-field: null rate, cardinality, value distribution.
PKFK→data_profiles
id UUID · field_name · null_rate · unique_count · top_values JSONB · min/max

quality_rules

Configurable quality rules. Add via API, no redeploy needed.
PKFK→data_sources
id UUID · dimension · rule_code · severity · config JSONB · active

quality_runs

Links a job to its quality evaluation session.
PKFK→ingestion_jobsFK→data_sources
id UUID · total_records · total_issues

quality_scores

Per-dimension score + overall + letter grade.
PKFK→quality_runs
id UUID · dimension · score · overall_score · grade (A/B/C/D/F)

quality_issues

Every violation: one row per field that failed one rule.
PKFK→quality_runsFK→staged_records
id UUID · dimension · severity · field_name · field_value · message · status

conflict_records

Immutable record of "Source A said X, Source B said Y."
PKFK→data_sources×2
id UUID · entity_type · field_name · value_a · value_b · resolution_strategy · status
Canonical Store (canonical schema) — External Tools Read Here

canonical.customers

One row per real customer, merged from all sources.
PK
id UUID · external_refs JSONB · name · segment · primary_source · quality_score · has_conflicts

canonical.products

Unified product catalog.
PK
id UUID · external_refs JSONB · sku · name · category · unit_price · currency · active

canonical.salespersons

Unified sales reps.
PK
id UUID · external_refs JSONB · name · team · territory · active

canonical.regions

Static region lookup — not merged from sources.
PK
id UUID · name · country · zone

canonical.orders

Unified orders. FK chain to customers/salespersons/regions.
PKFK→customersFK→salespersonsFK→regions
id UUID · order_date · total_amount · currency · source_id · job_id · has_conflicts

canonical.order_line_items

Atomic sale unit. line_total = quantity × unit_price − discount.
PKFK→ordersFK→products
id UUID · quantity · unit_price · discount · line_total
Lineage (public schema)

data_lineage

Full audit trail: canonical record → source → transformations applied.
PKFK→data_sourcesFK→ingestion_jobsFK→staged_records
id UUID · canonical_id · canonical_type · transformations JSONB
